# Obsidian Pulse Theme

Obsidian Pulse is a professional, dark-only **Visual Studio Code theme**, blending Dracula, Flat, and Material design. Designed for long coding sessions, it is easy on the eyes with vibrant syntax highlighting and harmonious UI shades.

## Features

- Dark-only theme
- Green mint strings for a fresh, modern look
- Harmonious shades for workspace, sidebar, status bar, title bar, and tabs
- Vibrant and professional syntax highlighting
